pDHS-SVM:

A Prediction Method for Plant DNase I Hypersensitive Sites Based on Support Vector

 Read Me

The web-server iNuc-PseKNC was developed to predict nucleosome occupancy of H. sapiens, C. elegan and D. melanogaster . The overall accuracy of 86.27%, 86.90%, 79.97% were achieved respectively for the above spieces by using jackknife cross-validation. The results obtained by iNuc-PseKNC on various benchmark datasets used by the previous investigators for different genomes also indicated that the current predictor remarkably outperformed its counterparts. All benchmark data can be downloaded from the Data window of this web-server.

Cavent

(1) For each submission, the input sequences must be in FASTA format; i.e., each DNA sequence should start with a greater-than symbol (" > ") in the first column. The words right after the " > " symbol in the single initial line are optional and only used for the purpose of identification and description.

(2) If a query sequence contains any illegal characters, wrong results will be obtained.

  Data

The H. sapiens dataset contains 2,273 nucleosome sequences and 2,300 linker DNA sequences, which can be Downloaded  .

The C. elegan dataset contains 2,567 nucleosome sequences and 2,608 linker DNA sequences, which can be Downloaded  .

The D. melanogaster dataset contains 2,900 nucleosome sequences and 2,850 linker DNA sequences, which can be Downloaded  .
